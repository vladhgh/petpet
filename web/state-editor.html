Запущено через ./petpetctl.sh editor — правки сразу пишутся в states.json, и хук подхватывает их на следующем событии (анимация · статус · цвет/иконка для каждого состояния). Открыто как file:// → офлайн-режим: правки только в браузере (localStorage).
